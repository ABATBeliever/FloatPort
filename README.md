# FloatPort

**浮遊する常駐型ブラウザ**

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Version](https://img.shields.io/badge/version-1.0.0-green.svg)
![Version](https://img.shields.io/badge/Language-HSP3.7-yellow.svg)

FloatPort は、画面上に常駐・浮遊する小型の Web ブラウザです。  
最前面固定表示や半透明表示などの機能を活用し、作業中の検索・参照を素早く行うことを目的としています。

本ソフトウェアは Microsoft 社の **WebView2** 技術を使用しています。

![img1](https://abatbeliever.net/app/FloatPort/img1.png)

---

## 入手
[こちら](https://github.com/ABATBeliever/FloatPort/releases) からどうぞ。

## 主な特徴

- 最前面固定・半透明表示
- 小型で邪魔にならないブラウザ
- ショートカットキーまたはマウスの動きで呼び出し可能
- 一般的な Web サイトを Edge 同様に閲覧・検索可能
- インストール不要、プロファイル管理対応
- 更新通知機能あり

---

## 動作環境

### 必須

- Windows 10 22H2 / Windows 11（x86_64）
- Microsoft Edge WebView2 Runtime

※ WINE / Proton / ReactOS などの互換環境はサポートしません。

### 推奨

- 400MB 以上の空き容量
- Wi-Fi 環境
- 解像度 640 × 480 以上（推奨 1280 × 720 以上）

---

## セットアップ方法

1. zip ファイルを任意のフォルダに展開してください  
2. `FloatPort.exe` を実行すると起動します  

※ レジストリの変更は行いません  
※ アンインストールはフォルダごと削除してください  

---

## 使用方法

- 起動後、アドレスバーに URL または検索語句を入力してください  
  - Ctrl キーを押しながら検索すると、URL として移動します
- 通常の Web ブラウザと同様にページ閲覧が可能です
- 最前面表示・透過設定・ショートカットキーはアプリ内で切り替えられます  

※ 既定のショートカットキーは **Ctrl + Shift** です。

---

## プロファイル設定

初回起動時に `FloatPort.exe.WebView2` フォルダが生成されます。  
このため、初回起動は時間がかかることがあります。

FloatPort はこのフォルダを参照し、以下の情報を保存します。

- 履歴
- Cookie
- 各種 Web データ

初期化したい場合は、このフォルダを削除してください。  
フォルダをバックアップすることで、状態を保存できます。

## 互換性

プロファイルは、当方が開発する  
**「InternetStroller」シリーズの「Memoria」以降**と互換性があります。

---

## 制限事項・既知の問題

- DRM 保護のある Web サイトは動作しません
- IE 互換モードを必須とするサイトは動作しません
- 高度な Web アプリの動作は保証されません
- 個人開発ソフトウェアです。MIT ライセンスのもと、現状のまま提供されます
- 起動中の拡大率変更や画面の横断は、表示上の問題を引き起こす可能性があります

---

## 更新について

起動時に更新確認サーバへ接続し、アップデートを確認します。  
無効にしたい場合は `hspinet.dll` を削除してください。

---

## エラー一覧と対処法

| エラーコード | 内容 |
|------------|------|
| 0x00 | 画面サイズの把握に失敗 |
| 0x01 | マルチディスプレイ情報の把握に失敗 |
| 0x10 | wv2s.dll が見つかりません |
| 0x11 | WebView2 の展開に失敗 |
| 0x12 | ショートカットキー登録が無効 |
| 0x20～0x24 | 更新確認に失敗（使用可能） |
| 0x90 | 開発バージョン不足 |
| 0x91 | 未知の重大なエラー |

※ 改善しない場合は、発生状況とともに開発者までご連絡ください。
