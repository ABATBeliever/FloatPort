■ 概要
FloatPort は、画面上に常駐・浮遊する小型の Web ブラウザです。
最前面固定表示や半透明表示などの機能を活用し、作業中の検索・参照を素早く行う
ことを目的としています。

本ソフトウェアは Microsoft社の WebView2 の技術を使用しています。
==============================================================================
■ 主な特徴
・最前面固定・半透明表示
・小型で邪魔にならないブラウザ
・ショートカットまたはマウスの動きで呼び出し
・一般的な Web サイトをEdge同様に閲覧・検索可能
・インストール不要、プロファイル管理が可能
・更新通知
==============================================================================
■ 動作環境
【必須】
・Windows 10 22H2 / Windows 11 (x86_64)
・Microsoft Edge WebView2 Runtime
※WINE,Proton,ReactOS,及びその他の互換システムはサポートしません。
【推奨】
・400MB以上の空き容量
・Wi-Fi環境
・解像度 640 × 480 以上（推奨 1280 × 720 以上）
==============================================================================
■ セットアップ方法
1. zip ファイルを任意のフォルダに展開してください
2. FloatPort.exe を実行すると起動します
※ レジストリの変更は行いません
※ アンインストールはフォルダごと削除してください
==============================================================================
■ 使用方法
・起動後、アドレスバーに URL または検索語句を入力してください
　(Ctrlキーを押しながら検索するとURLとして移動できます)
・通常の Web ブラウザと同様にページ閲覧が可能です
・最前面表示や透過設定、ショートカットキーはアプリ内の操作で切り替えられます
※ショートカットキーは、既定ではCtrl+Shiftです。
==============================================================================
■ プロファイル設定
初回起動をすると、FloatPort.exe.WebView2フォルダが生成されます。
このため、初回起動は時間がかかることがあります。
FloatPortはこのフォルダを常に参照し、履歴やCookieなどを保存します。
初期化する場合はこれを削除してください。
フォルダをバックアップすると、その時点での状態を保存できます。

■ コマンドショートカットについて
コマンドショートカットとは、URLバーに特定のコマンドを入れ、ctrl+Enterで実行
できる拡張機能です。
◇home
　起動時のページに移動します
◇exit
　ブラウザを正常に終了します
◇restart
　ブラウザを再起動します
◇reset
　ブラウザを再起動し、セッションをバックアップし、新たに生成します。
　閲覧履歴やCookieは初期化されますが、バックアップから手動で復元できます。
　これには、初回起動時と同様の時間がかかることがあります。
　設定は初期化されません。設定を初期化したい場合はConfig.iniを削除

■ エラーの一覧と対処法
◇0x00
　画面サイズの把握に失敗しました。OSのバージョン更新や再起動を試し、改善
　されない場合は開発にご一報ください。
◇0x01
　マルチディスプレイ情報の把握に失敗しました。OSのバージョン更新や再起動を試し
　改善されない場合は開発にご一報ください。
◇0x02
　hspext.dllが存在しないため、リセット処理ができませんでした。
　hspext.dllを再配置することで解決します。
◇0x03
　リセット処理に失敗しました。
　ほかにアクティブなFloatPortプロセスがある場合は、終了してください。
◇0x10
　wv2s.dllが見つかりません。同じフォルダ上にあることを確認してください。
　無い場合は、再ダウンロードしてください。
◇0x11
　WebView2の展開に失敗しました。アプリのあるディレクトリへの書き込み権限が
　あることを確認ください。
◇0x12
　ショートカットキーの登録が無効です。2つまたは3つを選択して下さい。
◇0x20～0x24
　更新確認に失敗しました。ソフトは使用可能です。
　hspinet.dllを削除すると、エラーが発生しなくなります。
◇0x90
　開発バージョンが要求に達していません。このエラーは、通常発生しません。
◇0x91
　未知の重大なエラーです。表示される情報と発生状況を、開発にご一報ください。
