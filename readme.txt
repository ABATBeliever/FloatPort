■ 概要
FloatPort は、画面上に常駐・浮遊する小型の Web ブラウザです。
最前面固定表示や半透明表示などの機能を活用し、作業中の検索・参照を素早く行う
ことを目的としています。

本ソフトウェアは Microsoft社の WebView2 の技術を使用しています。
==============================================================================
■ 主な特徴
・最前面固定表示が可能
・ウィンドウの半透明表示に対応
・小型で邪魔にならないブラウザ
・一般的な Web サイトをEdge同様に閲覧・検索が可能
・インストール不要、プロファイル管理が可能
==============================================================================
■ 動作環境
【必須】
・Windows 10 / 11
・Microsoft Edge WebView2 Runtime
※WINE,Proton,ReactOS,及びその他の互換システム上では動作しません。
【推奨】
・400MB以上の空き容量
・Wi-Fi環境
・解像度 640 × 480 以上（推奨 1280 × 720 以上）
==============================================================================
■ セットアップ方法
1. zip ファイルを任意のフォルダに展開してください
2. FloatPort.exe を実行すると起動します
※ レジストリの変更は行いません
※ アンインストールはフォルダごと削除してください
==============================================================================
■ 使用方法
・起動後、アドレスバーに URL または検索語句を入力してください
・通常の Web ブラウザと同様にページ閲覧が可能です
・最前面表示や透過設定はアプリ内の操作で切り替えられます
※ショートカットキーは、既定ではCtrl+Shiftです。
==============================================================================
■ プロファイル設定
初回起動をすると、FloatPort.exe.WebView2フォルダが生成されます。これには時間が
かかることがあります。
FloatPortはこのフォルダを常に参照し、ここに履歴やCookieなどを保存します。
初期化する場合はこれを削除してください。
フォルダをバックアップすると、その時点での状態を保存できます。
==============================================================================
■ 互換性
プロファイルは、当方が開発する InternetStroller Memoria/Syncria と互換性が
あります。
==============================================================================
■ 制限事項・既知の問題
・DRM保護のあるウェブサイトや、IE互換モードを必須とするサイトは動作しません。
・高度な Web アプリの動作は保証されません。
・あくまでも個人開発のソフトウェアです。MITライセンスのもと、現状のまま提供
されます。
